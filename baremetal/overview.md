# 裸金属服务器

裸金属服务器（Bare Metal Server）是 Submodel.ai 提供的专用物理服务器租用服务。用户独占一台物理服务器的全部资源，享受零虚拟化损耗的极致性能，适合对计算性能、延迟或硬件控制有极高要求的工作负载。

## 主要特性

- **独享硬件**：独占物理服务器的全部 CPU、内存、GPU 及存储资源，无资源争抢
- **零虚拟化开销**：直接运行在裸机上，无 hypervisor 层，性能损耗极低
- **完整硬件控制**：可直接访问 NVIDIA GPU 驱动、CUDA 及底层硬件接口
- **高度可定制**：支持自定义操作系统配置、驱动版本及系统软件栈
- **物理隔离**：独享物理机保障，适合对数据安全和合规有严格要求的场景

## 适用场景

- 大规模 AI 模型训练，需要最大化 GPU 性能
- 对延迟极度敏感的实时推理服务
- 科学计算、分子动力学、物理仿真等高性能计算任务
- 需要硬件级控制（如 RDMA、InfiniBand 网络）的分布式训练
- 对虚拟化环境有合规限制的企业用户

## 与 Pod 的区别

| 特性 | 裸金属服务器 | Pod（GPU 容器） |
|------|------------|----------------|
| 资源独享 | 完全独享物理机 | 与其他用户共享宿主机 |
| 虚拟化 | 无 | 基于容器（Docker） |
| 性能 | 最高（无损耗） | 接近原生（极低损耗） |
| 启动时间 | 较长（分钟级） | 较快（秒级） |
| 灵活性 | 固定配置 | 灵活选择 GPU 数量 |
| 适合场景 | 超大规模训练、高安全要求 | 通用 AI/ML 工作负载 |

## 开始使用

1. 登录 [Submodel.ai 控制台](https://submodel.ai)
2. 在左侧导航栏中选择 **裸金属（Bare Metal）**
3. 浏览当前可用的裸金属服务器配置
4. 选择符合需求的服务器规格（CPU、内存、GPU 型号、存储配置）
5. 确认租用并提交订单
6. 等待服务器配置完成后，通过控制台获取 SSH 连接信息

## 访问与管理

裸金属服务器通过 SSH 进行访问和管理：

- 服务器创建完成后，控制台会显示 SSH 连接信息（IP 地址、端口）
- 使用你账户关联的 SSH 密钥进行认证
- 获得 root 权限，可完全自定义服务器环境
