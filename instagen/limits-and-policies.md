# 限额与政策

## 速率限制

- **并发请求**:每个 IP 1000
- **无每日上限**:请求和 token 数量不受限
- **更高限额**:企业需求请联系我们

## 使用政策

- **合理使用(Fair Use)**:仅限正当的 AI 交互
- **禁止滥用**:避免过量请求
- **监控**:主动的滥用检测

## 申请更高限额

联系我们时请提供:
- 使用场景描述
- 预期流量模式
- 业务理由

## 最佳实践

- **批量请求**:合并相关调用
- **高效 prompt**:优化 token 用量
- **客户端限流**:实现自己的速率管理

## 联系方式

- **邮箱**:support@submodel.ai
- **主题**:Rate Limit Increase Request
