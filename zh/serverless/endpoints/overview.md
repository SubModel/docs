# 概述

Worker 会在云端无缝执行你的代码。

## 核心特性

* **完全托管的执行环境**:SubModel 负责所有底层基础设施,确保你的代码按需运行,无需任何服务器搭建或维护。
* **自动伸缩**:平台会根据工作负载动态伸缩你的函数,优化资源利用率。
* **灵活的语言支持**:借助 SubModel SDK,你可以使用多种编程语言编写函数,兼具灵活性与易用性。
* **无缝集成**:SubModel 为你上传的代码提供一个 Endpoint,让你能够轻松地将 Handler 集成到任何应用组件中。

## 快速开始

开始使用 SubModel Worker:

1. **编写你的函数**:使用受支持的编程语言开发你的 Handler。
2. **部署到 SubModel**:将你的 Handler 上传到 SubModel 平台。
3. **集成并执行**:利用提供的 Endpoint 将函数集成到你的应用中,并开始执行。
