# FAQ

### Hosting 常见问题

#### 1. 问：如何确保云服务合规性？

答：我们提供一种云平台服务模式，通过计算资源池提供混合云服务。实施了客户隔离和敏感操作限制。

#### 2. 问：收益模式和分配方式是怎样的？

答：我们采用三种分配模式：Serverless、Pod 和闲置挖矿。假设基础收益为 X，则 Serverless 模式产生 2X 收益；Pod 模式产生 X 收益；闲置挖矿模式产生 0.5X 收益。这三种模式确保设备收益稳定可靠。

#### 3. 问：资源托管和释放的方式有哪些？

答：我们目前采用灵活托管，允许随时撤出，便于按需协调设备的托管与释放。

#### 4. 问：如何确保收益的清晰和分成？

答：我们使用 AB 账户进行账单展示。A 账户为公司账户，可查看所有收益和设备状态。B 账户为个人账户，既可查看公司账户的所有数据，也可查看个人利润收益和详细账单信息。

### 常见问题排查

| 问题             | 解决方案                                 |
| -------------- | ------------------------------------ |
| Agent 启动失败     | 检查 Docker 服务状态和 API key 权限。          |
| 节点状态显示 Offline | 验证网络连通性（telnet api.submodel.ai 443）。 |
| 任务调度失败         | 确认资源标签匹配以及资源配额充足。                    |
| GPU 无法识别       | 重新安装 NVIDIA 驱动和 CUDA 工具包。            |
