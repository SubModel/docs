# InstaGen 隐私政策

本隐私政策专门适用于 SubModel.ai 平台内的 InstaGen 服务。

## 数据收集与使用

### 我们收集的信息

* **Access Key 信息**:名称、描述及使用统计
* **model 使用数据**:请求次数、token 消耗量及费用
* **API 交互日志**:时间戳及基本的请求元数据
* **账户信息**:计费与用量跟踪数据

### 我们如何使用你的数据

* **提供服务**:处理你的 AI model 请求
* **计费与用量**:计算费用并跟踪消耗
* **服务改进**:分析使用模式以提升性能
* **安全保障**:监测未经授权的访问与滥用行为

## AI model 的数据隐私

### 不保留内容

* **用户 prompt**:我们不会存储你的输入 prompt 或请求
* **model 输出**:生成的内容在交付后不予保留
* **对话历史**:聊天会话不会被存储
* **文件上传**:仅作临时处理,不作永久存储

### 实时处理

* 所有数据均实时处理,以便即时响应
* 不缓存或存储任何数据以供日后使用
* 处理仅在请求处于活动状态期间进行

## 数据安全

### 访问控制

* 加密的 API key,用于安全认证
* 安全的传输协议(HTTPS/TLS)
* access key 的轮换与管理
* 速率限制以防止滥用

### 基础设施安全

* 企业级数据中心安全防护
* 定期的安全审计与更新
* 符合行业安全标准

## 第三方 model

### model 提供方隐私

* 部分 model 由第三方提供方提供
* 各提供方可能有其自身的隐私政策
* 我们不会与 model 提供方共享你的数据
* 你的交互内容对 SubModel.ai 保持私密

### 数据处理

* model 提供方仅接收处理所必需的数据
* 不共享任何个人信息
* 处理在安全的环境中进行

## 你的权利

### 数据访问

* 查看你的使用统计与计费信息
* 访问你的 access key 管理
* 导出你的用量数据以供分析

### 数据控制

* 创建、编辑和删除 access key
* 监控并控制你的用量
* 随时禁用 access key

### 数据删除

* 永久移除 access key
* 请求删除账户数据(须遵守法律要求)
* 依法保留计费记录

## 合规

### 监管标准

* 遵守适用的数据保护法律
* 定期更新隐私政策
* 透明的数据处理实践

### 国际用户

* 数据处理可能发生在不同司法管辖区
* 遵守当地的隐私法规
* 为数据传输提供适当的保障措施
