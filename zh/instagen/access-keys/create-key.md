# 创建 Access Key

## 创建你的第一个 key

1. 进入 **Account Settings** → **InstaGen Access Key**
2. 点击 **"Add New Access Key"**
3. 填写名称和描述
4. 复制你的 key

## key 命名示例

* `Production-LLM-API`
* `WebApp-Integration`
* `Testing-Environment`

## 安全注意事项

* 将 key 存储在环境变量中
* 切勿提交到版本控制
* 使用安全的密钥管理方案
* 定期轮换 key
