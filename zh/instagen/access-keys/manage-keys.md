# 管理 Access Key

## key 操作

* **启用/禁用(Enable/Disable)**:切换 key 状态
* **编辑(Edit)**:更新名称和描述
* **重置(Reset)**:生成新的 token 值
* **删除(Delete)**:永久移除 key

## 安全最佳实践

* **key 轮换**:每 90 天轮换一次
* **访问控制**:仅限必要人员使用
* **监控**:定期检查用量模式
* **规范化**:使用具描述性的名称

## 故障排查

* **key 无法使用**:检查是否处于活动状态
* **触发速率限制**:监控用量模式
* **认证错误**:核对 key 的格式与状态
