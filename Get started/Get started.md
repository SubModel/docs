# 快速开始

欢迎使用 Submodel.ai,一个专为人工智能、机器学习应用以及广泛计算需求打造的多功能云计算平台。

## 创建账户

首先创建你自己的 Submodel.ai 账户,以便管理和访问各类资源。

[在此注册](https://submodel.ai/#/login?redirect=%2Fdashboard) 创建账户。注册完成后,登录即可开始使用我们的服务。

## 为账户充值

在部署资源之前,你需要为账户充值。Submodel.ai 接受信用卡或加密货币支付。

[在此充值](https://submodel.ai/#/recharge)。

有关支付方式和账单的更多详情,请访问我们的 [账单信息](Billing%20information.md) 页面。

## 运行你的资源

账户充值完成后,你就可以部署计算资源了。请按照这份分步指南启动你的第一个 pod。
